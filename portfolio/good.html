I created a blogging/social network platform that aggregates posts from other social networks with a team of four other developers. The project is like a simpler version of diaspora.org. The project is RESTful and well documented.

Project: Memory Scanner

September 2016-October 2016

I worked with a partner to write a C program that scans all the pages in memory and returns a list of blocks of memory that contain contiguous pages with the same permissions (read only, read-writeable, and inaccessible). We also wrote three driver programs to test and demonstrate the memory scanning functions.
